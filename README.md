Overview

This project demonstrates an end-to-end data analytics workflow using Pandas, SQL, and Power BI. It focuses on analyzing real-world customer shopping behavior data to generate actionable insights for business decisions.

Tools & Technologies

Pandas: Data loading, cleaning, preprocessing, and feature engineering.

MySQL: Storing cleaned data and performing structured queries for analysis.

Power BI: Interactive dashboards and visualizations for insights presentation.

Data Cleaning

Handled missing values using category-wise median to avoid bias.

Standardized column names for consistency and SQL compatibility.

Removed redundant columns for cleaner datasets.

Feature Engineering

Created age segments using quartile-based binning to analyze customer behavior across age groups.

SQL Analysis

Used SQL to analyse aggregate, compare data efficiently

Examples include top discounted items, customer segmentation, and purchase trends by shipping type.

Power BI Visualization

Connected directly to MySQL for real-time insights.

Created KPI cards, charts, drill-downs, and conditional formatting for interactive analysis.

Filters and slicers enable dynamic exploration across categories, age groups, and gender.

Outcome

End-to-end solution from raw data to actionable insights.

Provides a clear workflow for customer analytics projects and interview discussions.
